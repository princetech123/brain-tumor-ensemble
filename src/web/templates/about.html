About the Neuro-Ensemble Framework

Clinical background, model architecture details, and project documentation.

Clinical Significance

Early detection and classification of brain tumors are critical steps in clinical neuro-oncology. MRI is the standard imaging modality due to its superior soft tissue contrast. However, manual interpretation is time-consuming and subject to inter-observer variability.

This framework serves as a computer-aided diagnostic (CAD) utility. By combining predictions from multiple deep learning models using self-attention, it achieves higher diagnostic sensitivity and specificity, providing clinicians with a reliable second opinion.

Clinical Disclaimer: This system is a research-grade software project and is not FDA approved. All diagnostic predictions should be verified by a board-certified radiologist.

Model Architecture

The framework implements a two-stage training paradigm combining convolutional neural networks (CNNs) and transformer models:

  • ResNet50: A deep residual network that uses skip connections to prevent gradient vanishing, yielding robust localized spatial feature representations ($2048$ dimensions).
  • EfficientNetB3: An optimized architecture scaled using compound coefficient scaling (balancing network depth, width, and resolution) to extract high-efficiency features ($1536$ dimensions).
  • Vision Transformer (ViT-B16): A self-attention architecture that splits images into $16\times16$ patches and models global relationship contexts, yielding a $768$-dimensional classification vector.

The extracted features are projected to a uniform $512$ embedding dimension, stacked, and fused using a multi-head self-attention layer to weigh each backbone model's representations dynamically.

Explainable AI (XAI) Methods

Grad-CAM (Gradient-Weighted Class Activation Mapping)

For CNN backbones, we extract activations from the final convolutional layers. By computing the gradients of the tumor classification score with respect to these activations, we calculate the relative importance of each channel. The positive linear combination yields a heatmap displaying the exact spatial structures the model used to identify a tumor.

Transformer Attention Maps

For the Vision Transformer, we hook into the final self-attention encoder layer. We compute the dot-product similarity between the classification token (which aggregates the overall sequence representation) and the $196$ patch tokens. Reshaping this similarity vector into a $14\times14$ grid yields a direct mapping of the global attention field.
